# E2E Chat Demo

WebSocket 기반 실시간 채팅에 브라우저 간 키 교환과 메시지 암호화를 붙인 발표용 데모입니다.

## 기능

- 같은 방에 들어온 사용자끼리 실시간 채팅
- ECDH 공개키 교환
- AES-GCM 메시지 암호화
- 서버가 보는 암호문 패널 표시
- Render 무료 Web Service 배포 설정 포함

## 로컬 실행

```bash
npm install
npm start
```

실행 후 `http://localhost:3000`으로 접속합니다.

## Render 무료 배포

1. 이 폴더를 GitHub 저장소에 올립니다.
2. Render에서 `New` -> `Web Service`를 선택합니다.
3. GitHub 저장소를 연결합니다.
4. 설정은 `render.yaml`이 자동으로 잡아줍니다.
5. 배포가 끝나면 Render가 만든 주소로 접속합니다.

무료 플랜은 접속이 없으면 서버가 잠들 수 있어 첫 접속이 조금 느릴 수 있습니다.

## 발표 설명

이 데모에서 서버는 메시지를 복호화하지 않고 중계만 합니다. 브라우저는 입장할 때 공개키를 교환하고, 각 상대와 공유 암호키를 만든 뒤 메시지를 암호화해서 전송합니다. 서버 패널에는 평문 대신 암호문과 IV만 표시됩니다.
